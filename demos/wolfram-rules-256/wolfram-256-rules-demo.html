Elementary Cellular Automata

Elementary cellular automata are the simplest class of one-dimensional cellular automata. Each cell has two possible states (0 or 1), and the rule for updating the state of a cell depends only on the current state of the cell and its two neighbors.

Since there are 2³ = 8 possible configurations for a cell and its neighbors, and each configuration can map to either 0 or 1, there are 2⁸ = 256 possible elementary cellular automaton rules, numbered from 0 to 255.
